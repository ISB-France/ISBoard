# Guide : Créer et remplir un entretien

## Qu'est-ce qu'un entretien ?

Un entretien est un bilan annuel ou professionnel entre un collaborateur et son manager. Il se compose de :

- **Questions/Réponses** basées sur un template
- **Compétences** actuelles et futures
- **Objectifs** passés et futurs
- **Bilans** (collaborateur + manager)
- **Signatures** électroniques
- **PDF** généré automatiquement

---

## 1. Créer un entretien

### Via une campagne (recommandé)

1. **"Gestion" → "Campagnes" → "Nouvelle Campagne"**
2. Remplir :
   - **Nom** : ex. "EAP 2026 - Q2"
   - **Type** : Évaluation, Professionnel, etc.
   - **Dates** : début / fin
3. **"Sauvegarder"**
4. Ouvrir la campagne → **"Générer Entretiens"**
5. Sélectionner les collaborateurs (tous, par site, ou par fonction)
6. **"Générer"** → un entretien est créé pour chaque collaborateur

### Via création directe

1. **"Nouveaux Entretiens"**
2. Sélectionner une **Campagne** existante ou en créer une
3. Sélectionner les **Collaborateurs**
4. Un entretien est créé pour chacun (statut initial : "Créé")

---

## 2. Remplir un entretien

### Onglet "Questions"
- Répondre aux questions du template
- Les réponses sont sauvegardées automatiquement

### Onglet "Compétences"
- Compétences actuelles : sélectionner les compétences, niveau requis, appréciation collaborateur + manager, remarques
- Compétences futures : compétences à développer (N+1), définition manager

### Onglet "Objectifs"
- Objectifs passés (N-1) : évaluer le niveau de réalisation (%), ajouter commentaires
- Objectifs futurs (N+1) : définir les nouveaux objectifs, thèmes, dates de réalisation

### Bilans
- **Bilan collaborateur** : auto-évaluation
- **Bilan manager** : évaluation par le manager

---

## 3. Signer l'entretien

1. **Collaborateur** → "Signer" → dessiner la signature → "Valider"
2. **Manager** → "Signer" → dessiner la signature → "Valider"
3. Statut passe à "Signé (Complètement)" et l'entretien est verrouillé

---

## 4. Générer le PDF

1. Cliquer **"Générer PDF"**
2. Le PDF est créé et envoyé par email
3. Disponible dans OneDrive : `/Entretiens/[ANNÉE]/[COLLABORATEUR]/`

---

## Cycle de vie d'un entretien

```
Créé → En attente (avant réunion) → En cours (réunion) →
Signé (partiellement) → Signé (complètement) → Archivé
```

Correspondance statuts :
| Statut | Signification |
|---|---|
| À faire | Entretien créé, en attente |
| En cours | Réunion en cours |
| Terminé | Entretien finalisé |
